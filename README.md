# 🐦 Flappy Bird Clone (HTML5 Canvas)

A modern, feature-rich **Flappy Bird clone** built using **HTML5 Canvas, CSS, and Vanilla JavaScript**.  
This project replicates the original gameplay while adding smooth animations, dynamic difficulty, and a polished UI experience.

---

## 🎮 Live Features

- 🐤 Smooth bird physics (gravity + jump mechanics)
- 🧱 Procedurally generated pipes with random gaps
- 📈 Dynamic difficulty scaling (speed + gap adjustment)
- 🌙 Day / Night environment switching
- 🔊 Built-in sound system using Web Audio API
- 💯 Score + High Score system (stored in localStorage)
- ⏸ Pause / Resume functionality
- 📱 Mobile touch + desktop keyboard controls
- 🎨 Retro pixel-style UI with animations
- 🖥 Fully canvas-rendered gameplay


🛠️ Tech Stack
HTML5 Canvas

CSS3 (Retro UI Design)

Vanilla JavaScript (ES6)

Web Audio API
